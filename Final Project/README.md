# Personal Expense Tracking System

1. Project Description:
    This is a simple Java console-based application that helps users track their income and expenses. It allows users to categorize spending, set a budget limit, and view financial summaries with basic behavioral insights.

---

2. Features:   
    - Add income entries
    - Add expense entries with categories (Food, Transport, Entertainment, Other)
    - Set and update budget limit
    - Track total income and expenses
    - View spending summaries
    - Simple behavioral insights based on spending patterns
    - Budget warning system

---

3. How to Run:
    - Navigate to the project folder
    - Open project folder
    - Compile the program
    - Run and debug the program

---

4. Project Structure:
    - ExpenseTrackingSytem.java → Main menu and user interaction
    - BudgetTracker.java → Core logic and calculations
    - Included a readme file (this file)
